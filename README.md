# GestionCasiers

**GestionCasiers** est un projet personnel pour une première mise en application mon auto-apprentissage du langage C#.

### Techno:

* C#
* .NET 10.0.101

Le modèle du projet est une application console, simulant une interaction utilisateur sur un bloc de casiers connectés.

### Fonctionnalités envisagées:

* création de casiers
* affichage des informations des casiers (taille, emplacement, disponibilité, état ouvert/fermé)
* dépôt dans un casier disponible
* retrait d'objets d'un casier occupé
* ouverture/fermeture sécurisée par code personnalisé
